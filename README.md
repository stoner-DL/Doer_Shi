## 文件结构
- `MnistProcessing.py`：处理MNIST数据集的脚本。
- `calculate.py`：执行计算任务的脚本。
- `main.py`：主程序文件。
- `myNNnetwork.py`：神经网络模型定义文件。
- `randomtest.py`：随机测试脚本。
- 'mnist':训练和测试的数据集
